Tool1 — AI物件査定
tools/1-ai-satei.html ｜ 物件情報を入力すると売却概算価格を自動計算

実装済み機能

試算
売却コスト試算カード
仲介手数料・印紙税・登記費用を自動計算。「手元に残る目安」を大きく表示して現実的な資金計画をサポート。
情報
手取り活用プラン（手取り後の投資シナリオ）
売却後の純手取りを使った4シナリオ（定期預金・投信・不動産再投資・ローン繰り上げ）の年間リターン比較。
情報
売却プロセスタイムライン
査定→媒介契約→広告→内覧→契約→引渡しまでの5ステップと期間目安を図で表示。初めての売却でも安心。
試算
譲渡所得税シミュレーター
取得費・保有期間・3,000万円特別控除に対応。課税譲渡所得→税率→概算税額を一画面で確認。
UI
物件グレード評価カード（S/A/B/C/D）
立地・築年数・階数・駅距離・方角の5軸100点スコアでグレードを自動評価。
情報
売却時期カレンダー
月別の成約需要（春需要・秋需要）を色付きカレンダーで表示。最適な売り出し時期をひと目で確認。
情報
近隣エリアとの坪単価比較パネル
ユーザーのエリアと周辺エリアの坪単価を棒グラフで比較。自分の物件価格の妥当性がわかる。
情報
買主の視点カード
物件条件から「アピールポイント」「懸念点」「見込み買主層」を自動生成。売り主が見落としがちな買主目線の情報を提供。
UI
査定履歴のスパークライン（価格推移グラフ）
直近10件の査定価格をSVGラインチャートで表示。前回比の騰落率を色分け表示。
UI
相談前確認チェックリスト（インタラクティブ）
「権利書の確認」「ローン残高把握」など10項目をタップでチェック。進捗バー付き。
CTA
SNSシェアボタン
査定結果をネイティブシェア（iOS/Androidのnavigator.share）またはクリップボードコピーで共有。
情報
よくある質問FAQアコーディオン
「本当にこの価格で売れる？」「ローンが残っている場合は？」など6問のQ&Aをタップで開閉。
情報（新）
周辺の成約事例カード
ユーザーの物件と条件が近い近隣3件の成約価格・坪単価・成約期間を表示。査定価格の妥当性を確認できる。
試算（新）
売却タイミング比較カード
「今」「3年後」「5年後」「10年後」に売った場合の想定価格・手取りを並べて比較。最適売却タイミングを診断。

Tool2 — 空き家活用診断
tools/2-akiya-hunter.html ｜ 空き家の状態チェックと活用提案

実装済み機能

試算
放置リスク試算カード
固定資産税・維持費・5年累計コスト・機会損失を自動計算。「放置するとどうなるか」を数字で提示。
試算
売る・貸す・放置の5年比較
固定資産税評価額から3択の5年間損益を試算。各選択肢のメリット・デメリット付き。
試算
建て替え vs フルリノベ 比較カード
費用・月間賃料・年収・回収期間を並べて比較。どちらがお得か一目瞭然。
試算
修繕費用概算カード
「軽微補修」「中規模リフォーム」「全面改装」「建て替え」の4段階費用目安を築年数別アドバイス付きで表示。
情報
補助金・支援制度カード
解体補助・リノベ補助・空き家バンク奨励金など6種類の支援制度を診断結果に応じて自動絞り込み表示。
情報
空き家市場性評価カード（5軸スコア）
築年数・立地・構造・権利関係・管理状態の5軸で「売りやすさ」を診断。
UI
チェックリスト進捗バッジ（n/12チェック）
診断チェックリストのリアルタイム進捗カウントと「全選択/全解除」トグルボタン。
CTA
印刷レポート生成
診断スコア・チェックリスト・リスク試算・3択比較・次のステップを含む印刷用レポートを新規ウィンドウで生成。
情報
同じような空き家オーナーの解決事例4件
「賃貸活用」「売却」「駐車場転用」「空き家バンク」のパターン別事例カード。「自分もできそう」という気づきを促進。
試算
解体費用かんたん試算
木造・鉄骨・RC別、延床面積から解体費用の目安を試算。

Tool3 — 売主・買主マッチング
tools/3-owner-direct.html ｜ 売主・買主を登録してAIスコアでマッチング

実装済み機能

スコア
マッチングスコア算出（0-100点）
価格・エリア・種別・面積・タイミングの5要素でスコアを計算。スコア別に色分け（緑/金/青）表示。
UI
売主カードに「マッチ人数バッジ」
スコア30以上の買主が何人いるかをリアルタイム表示。「🎯 3人マッチ」のような視覚的バッジ。
UI
「今すぐ成約チャンス」バナー
売主・買主ともにタイミングが「即決」の場合、マッチングカードに「🔥 今すぐ成約チャンス」バナーを表示。
UI
買主カードに「購入確度バッジ」
即決×キャッシュは「⭐ 即決可能」、どちらかを満たすと「🔥 高確度」バッジを表示。
試算
ローン試算カード
登録済み買主の平均予算 × 7金利パターンの月返済額を比較。30%ルール（月収の30%以下）のヒント付き。
情報
買主予算帯分布カード（バーチャート）
売主の価格に対して、登録済み買主の予算がどう分布しているかをバーグラフで表示。
UI
マッチングヒートマップ（売主別）
全売主の「最高スコア」と「マッチ件数」をカード形式でグリッド表示。どの物件が人気かひと目でわかる。
情報
要因別マッチング統計
「価格一致」「エリア一致」などの要因ごとのマッチング割合をバーグラフで表示。改善ポイントを提示。
CTA
提案レター自動生成
マッチングペアごとに買主向けの物件提案文を自動生成。メール送信またはクリップボードコピー対応。
CTA
物件広告文の自動生成
売主カードからSNS向け短文・掲載用詳細文を自動生成。クリップボードコピー対応。
試算
仲介手数料シミュレーター（刷新版）
法定上限の自動計算に加え、諸費用総合計（印紙税・登記費用含む）を表示。価格帯別の目安表付き。
UI
スコア下限・エリアフィルター
マッチングタブでスコア下限（30/50/70点）とエリアでフィルタリング可能。
🏢
Tool4 — 賃貸物件かんたん管理
tools/4-kanri-saas.html ｜ テナント・家賃・修繕・収益を一元管理

実装済み機能

UI
ダッシュボード KPIカード
今月の入金・修繕費・収支・回収率・前月比を4列カードでリアルタイム表示。
UI
満室収益ポテンシャルカード
現在の収入と「もし空室が埋まったら」の最大収益を並べて表示。グラデーションの進捗バー付き。
UI
家賃台帳 入金進捗バー（新）
選択した月の入金済み家賃合計・回収率・未回収額をリアルタイムの進捗バーで表示。色がステータス（緑/金/赤）で変化。
UI
テナント優良度スコアカード
入金遵守・入居期間・情報完備・ステータスの4軸でS/A/B/Cをスコアリング。テナント詳細モーダルに表示。
UI
連絡先クイックアクセスカード
テナントタブの先頭に全アクティブテナントの📞電話・✉️メールリンクをまとめて表示。すぐ連絡できる。
CTA
未入金テナントへの督促メール生成
未払いテナントの行に「督促メール」ボタン。メールアドレス登録済みならmailto:リンクで直接送信可能。
CTA
契約更新メール自動生成
更新期限90日以内のテナントの行に「更新メール」ボタン。テナント名・物件名・更新日・家賃を含む文章を自動生成。
試算
賃料値上げシミュレーター
値上げ率スライダーで月次・年次増収額をテナント別にリアルタイム試算。
試算
確定申告ヘルパーカード
年間収入・修繕費・課税所得概算・控除確認リスト・申告メモを自動生成。確定申告前の準備をサポート。
UI
物件別収益スタックバーグラフ
2棟以上の物件がある場合に、月次収益を物件別の積み上げバーグラフで表示。
試算
修繕管理KPIカード（年間費・完了率）
年間修繕費合計と完了率をKPIとして表示。長期修繕計画との連動。
情報
長期修繕計画（積立シミュレーター）
今後の大規模修繕を登録して、月次積立目標額を自動計算。「毎月いくら積み立てるべきか」が一目でわかる。

Tool5 — 不動産投資シミュレーター
tools/5-toushi-bunseki.html ｜ 詳細な投資収益分析とリスク評価

実装済み機能

UI
専門用語ツールチップ（NOI・CCR・DSCR・IRR）
「？」マークをタップすると専門用語の意味をポップアップ表示。初心者でも迷わない。
UI
初心者向け「かんたんモード」
入力項目を5つに絞ったシンプルビュー。ローン月額・表面利回り・月次CF概算をリアルタイムプレビュー付き。
計算
IRR（内部収益率）Newton-Raphson法
真の内部収益率をNewton-Raphson反復法で精密計算。投資の真の収益性を正確に評価。
UI
金利上昇シミュレーションカード
現在金利±2%の6パターンで月返済額と差分を比較。「金利が上がったらどうなる？」を事前確認。
情報
3シナリオ比較カード（ベスト/ベース/ワースト）
感度分析タブで楽観・基準・悲観シナリオのCF・利回りを並べて比較。
試算
CF安全マージン分析（損益分岐点）
「空室率があと何%上がるとCFゼロ？」「家賃があと何%下がると？」を自動計算。
UI
CSVダウンロードボタン（CFテーブル）
年次キャッシュフロー詳細表をCSV形式でダウンロード。Excelで詳細分析可能。
情報
投資タイプ診断クイズ（7タイプ）
5問クイズで「インカムゲイン型/キャピタル型/安全重視型」など7タイプを診断。自分に合った投資スタイルがわかる。
UI
物件比較テーブル（最大3件）
履歴タブで過去分析した物件を最大3件並べてDSCR・IRR・CCRなどを比較。
試算
繰り上げ返済シミュレーター
毎月または一括で追加元本返済した場合の利息削減効果・返済期間短縮を試算。
UI
CFレポート印刷・PDF保存
かんたんモード・通常モード両方で印刷用レポート生成。PDFとして保存可能。

全ツール共通 / index.html
全ツールに適用された共通改善・トップページ

実装済み機能

UI
削除確認ダイアログ（削除対象名表示）
「本当に削除しますか？」のダイアログに削除対象の名前を表示。誤削除を防止。
UI
必須項目バリデーション（赤ボーダー）
フォームの必須項目が未入力の場合に赤ボーダーとエラーメッセージを表示。
UI
localStorage容量超過時のエラー表示
ブラウザの保存容量が満杯のとき、ユーザーフレンドリーなエラーメッセージを表示。
UI
モバイル対応（375px・640px）
全ツールでスマートフォン（375px〜）での表示を最適化。ボタンサイズ・フォントサイズも調整。
CTA
Supabaseクラウド保存（全ツール統合）
ログイン時に全データをSupabaseクラウドDBに保存・同期。ログアウトしてもデータが消えない。
CTA
GASバックエンド連携（全ツール）
全ツールの利用データをGoogle Apps ScriptのスプレッドシートにリアルタイムでPOST送信。データ蓄積を自動化。
情報
index.html: 今日の不動産豆知識
12種類の豆知識を日付によってローテーション表示。毎日違うコンテンツでリピーターを促進。
UI
index.html: ログイン後マイデータパネル
ログイン後に全テーブルのデータ件数（物件・テナント・査定・投資・空き家・マッチング）をサマリー表示。
UI
index.html: 利用者の声セクション
3件のユーザーレビューカードでサービスの信頼性を向上。
情報
全ツール: OGP画像・JSON-LD構造化データ
SNSシェア時のプレビュー画像設定。GoogleのWebサイト認識向上のための構造化データ追加。
📌 このツールの目的
TERRA REALTYが「情報優位の立場」に立つために、一般ユーザーが繰り返し使いたくなるツールを作り続けています。
ユーザーがツールを使うたびに査定データ・投資判断・物件情報が蓄積され、サービスの価値が高まる仕組みです。
